## Uh oh, it's my turn, what do I do?

Send out an email asking for contributions from everyone.

* Clone this repository
* Run ```bundle install```
* Run ```rake new_post```
* Commit / push post
* Run ```rake setup_github_pages```
** You will be prompted for the git repo
* Run ```rake gen_deploy```
